## 写一个 作用域插槽 使用场景？

::: details
可以参考官网例子。

```html
<!-- <todo-list> -->
<ul>
  <li v-for="item in items">
    <slot name="item" v-bind="item"></slot>
  </li>
</ul>

<!-- 父组件 -->
<todo-list :api-url="url" :per-page="10">
  <template #item="{ body, username, likes }">
    <div class="item">
      <p>{{ body }}</p>
      <p>by {{ username }} | {{ likes }} likes</p>
    </div>
  </template>
</todo-list>
```
